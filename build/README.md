# Build v1.0.1

This directory contains the compiled MCP server ready for deployment.

## Files
- `index.js` - Main MCP server (compiled from TypeScript)
- `index.js.map` - Source map for debugging
- `index.d.ts` - TypeScript declarations
- `package.json` - Package configuration

## Installation
```bash
npm install
node index.js
```
